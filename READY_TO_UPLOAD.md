2. Open your "Unified Game Launcher" app
3. Go to "Store listings" → English (United States)

**Upload Package:**
- Scroll to "Packages" section
- Upload: `release/Unified Launcher 1.0.0.appx`

**Upload Store Logos (OPTIONAL but recommended):**

- Upload `store-assets/store-icon-300x300.png` (most important!)
- Upload other sizes if Partner Center asks for them

**Add Description:**
- Copy from `STORE_DESCRIPTION.md`
- Paste in "Description" field

4. Click "Save" at bottom
5. Click "Submit to the Store"

## 📸 Quick Screenshot Guide

### If App is Empty (No Games):
Don't worry! Take screenshots anyway:
1. Run `npm run dev`
2. Show the interface even without games
3. Or manually add 1-2 sample games first
4. Capture the main view, settings, free games browser

### Best Screenshots to Take:
1. **Main library** - Even if empty, shows interface
2. **Fullscreen mode** - Click fullscreen button first
3. **Free games** - Click "Free Games" in sidebar
4. **Settings** - Click gear icon, show themes
5. **Any game details** - If you have games

### How to Capture:
- `Windows + Print Screen` - Auto-saves to Pictures/Screenshots
- `Windows + Shift + S` - Snipping tool (select area)
- `Alt + Print Screen` - Active window only
