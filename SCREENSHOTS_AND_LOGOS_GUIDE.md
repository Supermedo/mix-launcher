# Screenshots & Store Logos Guide

## 📸 Required: Screenshots (CRITICAL)

You need **at least 1 screenshot** (recommended: 3-5) to submit to the Store.

### Screenshot Requirements:
- **Minimum size**: 1366 x 768 pixels
- **Recommended size**: 1920 x 1080 pixels (Full HD)
- **Maximum**: 30 files
- **Format**: PNG or JPG
- **File size**: Less than 50 MB each

---

## 📸 HOW TO TAKE SCREENSHOTS

### Method 1: Run the App and Capture

1. **Start the app**:
   ```bash
   npm run dev
   ```
   Or run the portable exe from `release/Unified-Launcher-Portable.exe`

2. **Navigate to different screens**:
   - Main library view (with games)
   - Fullscreen console mode (press fullscreen button)
   - Free games browser (click "Free Games" in sidebar)
   - Game details view (click on a game)
   - Settings screen (click gear icon)

3. **Take screenshots** using Windows:
   - Press `Windows Key + Print Screen` - Saves to Pictures/Screenshots
   - Or use `Windows Key + Shift + S` - Snipping tool (select area)
   - Or use `Alt + Print Screen` - Captures active window only

4. **Save screenshots** as:
   - `screenshot1.png` - Main library
   - `screenshot2.png` - Fullscreen mode
   - `screenshot3.png` - Free games
   - `screenshot4.png` - Game details
   - `screenshot5.png` - Settings/themes

### Method 2: If App Has No Games Yet

If you don't have games installed, I can help you:
1. Add some sample games manually
2. Or take screenshots showing the interface even without games
3. Use demo mode if available

## 🎨 STORE LOGOS (Optional but Recommended)

You need to create store logos from your existing `build/icon.png`:

### Required Sizes:

#### 1. **Store Display Icon** (Required for Windows 10/11)
- 1:1 App tile icon: **300 x 300** pixels
- 1:1 (150x150): **150 x 150** pixels
- 1:1 (71x71): **71 x 71** pixels

#### 2. **9:16 Poster Art** (Optional - for Xbox display)
- Small: **720 x 1080** pixels
- Large: **1440 x 2160** pixels

#### 3. **1:1 Box Art** (Optional)
- Small: **1920 x 1920** pixels
- Large: **2160 x 2160** pixels

---

## 🛠️ HOW TO CREATE STORE LOGOS

### Option A: Use Online Image Resizer (Easiest)

1. Go to: https://www.iloveimg.com/resize-image
2. Upload `build/icon.png`
3. Resize to each required size:
   - 300x300 → Save as `store-icon-300x300.png`
   - 150x150 → Save as `store-icon-150x150.png`
   - 71x71 → Save as `store-icon-71x71.png`
4. Keep aspect ratio, transparent background

### Option B: Use Windows Paint (Built-in)

1. Open `build/icon.png` in Paint
2. Click "Resize"
3. Select "Pixels"
4. Uncheck "Maintain aspect ratio"
5. Enter width: 300, height: 300
6. Save as PNG
7. Repeat for other sizes

### Option C: Use PowerShell Script (Automated)

I can create a script to auto-generate all required sizes!
